# Wealth Tracker

Track your entire wealth in one private desktop app — stocks, crypto, real estate, and more.

**Latest: v1.0.0**

## Download

[Download for Windows](https://github.com/Twan01/wealth-tracker/releases/latest) · [View all releases](https://github.com/Twan01/wealth-tracker/releases)

## Features

- Dashboard with net worth tracking across 7 asset classes
- What-If scenarios and Time Machine
- Risk analysis and stress testing
- AI-powered insights (Pro+AI tier)
- 100% offline — your data stays on your machine

## Install Guide

1. Download the `.exe` from the [latest release](https://github.com/Twan01/wealth-tracker/releases/latest)
2. Run the installer — if Windows SmartScreen appears, click **More info → Run anyway**
3. Complete the setup wizard (choose install path, accept defaults)
4. Launch **Wealth Tracker** from the desktop shortcut or Start Menu
5. The app starts with the Free tier — enter a license key in Settings to upgrade
