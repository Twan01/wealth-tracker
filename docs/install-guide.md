# Installing Wealth Tracker

## Download

1. Go to the [latest release](https://github.com/Twan01/wealth-tracker/releases/latest)
2. Download `Wealth-Tracker-Setup-X.Y.Z.exe` (the `.exe` file)

## Windows SmartScreen Warning

Because Wealth Tracker is a new application, Windows SmartScreen may show a warning:

1. When you see **"Windows protected your PC"**, click **"More info"**
2. Then click **"Run anyway"**

This is normal for new applications that haven't built up a reputation with Microsoft SmartScreen yet. The installer is code-signed but the certificate is new.

## Installation

1. Run the downloaded `.exe` file
2. Choose your installation directory (default: `C:\Program Files\Wealth Tracker`)
3. Click **Install**
4. If prompted by UAC (User Account Control), click **Yes**
5. Wait for installation to complete
6. Check **"Launch Wealth Tracker"** and click **Finish**

## First Launch

1. Wealth Tracker will start and show the Dashboard
2. Your data is stored locally in `%APPDATA%\Wealth Tracker`
3. No account or internet connection is required

## Entering a License Key

1. Go to **Settings** (gear icon in sidebar)
2. Under **License**, paste your license key
3. Click **Activate**
4. Features unlock immediately based on your tier (Free / Pro / Pro+AI)

## Automatic Updates

Wealth Tracker checks for updates automatically on launch. When a new version is available:

1. A dialog will ask if you want to update
2. Click **Update** to download in the background
3. Once downloaded, you'll be prompted to restart
4. Click **Restart Now** to install the update

You can disable auto-update checks in **Settings > Updates**.

## Uninstalling

Use **Windows Settings > Apps > Installed Apps > Wealth Tracker > Uninstall**, or run the uninstaller from the Start Menu.

Your data in `%APPDATA%\Wealth Tracker` is preserved after uninstalling.
